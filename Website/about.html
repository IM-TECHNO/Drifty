Open Source Software Developer

Hello people, I'm a passionate Software Developer and an open-source enthusiast building the super enhanced file downloading system called Drifty. Drifty is an open-source interactive File Downloader system built using Java. It takes the link to the file, the directory where it needs to be saved and the filename of the downloaded file as input and downloads it. It is currently available in CLI mode and the GUI version is under active development.
